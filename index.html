How to search

Type queries in plain English. The translator understands phrases like:

  • Colors: "red", "green and white", "azorius", "exactly blue"
  • Types: "creatures", "instants", "legendary dragons", "sagas"
  • Mana cost: "under 3 mana", "costs exactly 4", "cmc 2 or less"
  • Text: "with trample", "that draw cards", "with 'enters tapped'"
  • Power/Toughness: "power 5 or more", "1/1 creatures"
  • Format: "legal in commander", "modern legal", "in standard"
  • Rarity: "rare", "mythic", "common", "uncommon"
  • Sets: "from dominaria", "in modern horizons 3", "from mh3"
  • Special: "with flying", "with deathtouch", "that produce mana"

Tip: The translated Scryfall query is shown above the results. You can copy it or open Scryfall directly to refine further.

If a phrase isn't recognized, it's kept as a name search. You'll see a note about anything that didn't translate cleanly.
